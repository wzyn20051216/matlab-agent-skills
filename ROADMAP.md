# Roadmap

This roadmap tracks the path from a useful local skill suite to a serious open-source MATLAB agent workbench.

## v0.1

- MATLAB R2026a skill suite.
- Local deployment script.
- MATLAB batch runner.
- Toolbox inventory export.
- Numeric and Simulink smoke validation.
- GitHub Actions baseline workflow.

## v0.2

- Packaged `.skill` releases.
- More realistic evaluation prompts and benchmark reports.
- Public paper reproduction examples with small datasets.
- Better Simulink topology diff and model audit helpers.
- Code generation golden-output comparison templates.

## v0.3

- Self-hosted runner guide for licensed MATLAB/Simulink CI.
- Domain examples for control, robotics, signal processing, and vision.
- CI matrix for MATLAB releases where licensing allows.
- Example gallery with generated artifacts and validation summaries.

## Longer Term

- Agent-to-MATLAB orchestration patterns for large projects.
- Simulink model-based design review checklists.
- Reusable reproduction manifests for academic papers.
- Tooling to compare agent outputs against deterministic MATLAB test oracles.
